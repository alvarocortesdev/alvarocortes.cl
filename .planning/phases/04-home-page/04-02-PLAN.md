<task type="auto">
  <name>Task 1: Make Header responsive with mobile menu</name>
  <files>apps/web/src/components/Header.tsx</files>
  <action>
    Update Header for mobile responsiveness (RESP-03):

    **Mobile behavior (< 768px / md breakpoint):**
    - Logo/name still visible but potentially smaller
    - Navigation hidden behind hamburger menu icon
    - When hamburger clicked, show mobile nav (simple dropdown or slide-in)

    **Desktop behavior (>= 768px):**
    - Current layout preserved (logo left, nav right)

    **Implementation approach:**
    - Use React useState for menu open/close
    - Use Tailwind responsive classes (hidden md:block, etc.)
    - Simple hamburger icon (3 horizontal lines, can use divs)
    - Mobile nav: Dropdown below header or overlay

    **Example structure:**
    ```tsx
    import { useState } from 'react'

    export function Header() {
      const [menuOpen, setMenuOpen] = useState(false)

      return (
        <header className="fixed top-0 left-0 right-0 z-50 bg-neutral-900 border-b border-neutral-800">
          <div className="max-w-6xl mx-auto px-6 h-16 flex items-center justify-between">
            {/* Logo/Name */}
            <div className="flex items-center gap-2 md:gap-3">
              <div className="w-8 h-8 md:w-10 md:h-10 rounded-full bg-neutral-700 flex items-center justify-center text-xs md:text-sm font-bold">
                AC
              </div>
              <span className="text-base md:text-lg font-semibold">Alvaro Cortes</span>
            </div>

            {/* Desktop nav */}
            <nav className="hidden md:block">
              <a href="#" className="...">Inicio</a>
            </nav>

            {/* Mobile hamburger */}
            <button
              className="md:hidden p-2"
              onClick={() => setMenuOpen(!menuOpen)}
              aria-label="Toggle menu"
            >
              {/* Hamburger icon */}
            </button>
          </div>

          {/* Mobile nav dropdown */}
          {menuOpen && (
            <nav className="md:hidden bg-neutral-900 border-t border-neutral-800 px-6 py-4">
              <a href="#" className="block py-2">Inicio</a>
            </nav>
          )}
        </header>
      )
    }
    ```

    Keep it simple - no complex animations for v1.0.
  </action>
  <verify>
    - Header.tsx updated with useState import
    - Hamburger button visible on mobile (< 768px)
    - Desktop nav visible on desktop (>= 768px)
    - `bun run type-check` passes
  </verify>
  <done>Header responsive with mobile hamburger menu</done>
</task>
